PS:
        有的时候第一次安装有些插件可能会安装失败，启动的时候会有错误，比如
        `rainbow-blocks` 这个插件，这个时候就需要输入命令： 
        `M-x package-list-packages` 然后找到这个包，按 `d` ，然后按 `x`
        删除这个包，然后重启Emacs，会自动重新安装这个失败的包。一般情况下
        就没问题了～
